# TimeZonePicker – Native ELF Time Zone Selector for Siemens Mobile Phones

TimeZonePicker is an ELF application that replaces the built‑in Java time zone selection tool on Siemens mobile phones.

## Building
The SDK must be located at `../sdk` relative to the project root.
```bash
mkdir build && cd build
cmake ..
make
```
